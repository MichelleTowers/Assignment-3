Tools

Website Repository
Github was used by Michelle and Blayne for both the hosting of the applications website on Blayne’s Github and the hosting of the assignment website, meeting notes and artifacts on Michelle’s Github. The Github contributions on the assignment repository do not reflect on the participation of our group members as we chose to use external applications and websites.

Audacity was used by each team member to record their voice overs for the presentation podcast.

The Sims 4 was used by Michelle for the demonstration of our podcast visuals as it was the easiest option due to our limited time and members.

Adobe audition was used by Michelle to add all of the Audacity recordings and save them as one file. Also to edit out mistakes from any recording and to generate our Elon Musk robots voice.

Adobe premiere pro was used by Michelle to edit the Sims 4 gameplay and podcast audio together.

Google Docs was used by all team members to contribute to our drafts document, which held all information about the project across the whole time frame.

Adobe Photoshop was used by Blayne with the assistance from Michelle and Helia to adjust and edit images for personality tests in the team profiles section.

Invision was used by all team members to collaborate in creating website and application wireframes as evidence towards the artifacts section.

Canva was used by all team members to create application screen mockups, marketing posters, user flows, legal pdf, marketing pdf and the final document report.

TeleportHQ was used by Michelle and Blayne to create the website for the application, as well as the overall assignment.

App Demo Store was used by Blayne with the assistance from Michelle and Helia to create working screen flows through the application and its screen mockups.

Termly was used by Michelle to create the legal documentations.

MS Teams was used by all team members for communication and collaboration across the whole assignment period.

Visual Studio Code was used by Michelle and Blayne to attempt coding basic screens for the application and demonstrate some basic concepts of programming to Helia.

Youtube was used by Blayne via screen share to be the personal DJ playing motivational music during our collaborative sessions. Our motivational song of choice was decided by the group.
